## Overview

A capstone project covering the history, licensing, ethics, and practical shell scripting of Linux as an open-source operating system.

---

## Topics Covered

- **Origin of Linux** — Why Linus Torvalds created Linux in 1991 and the problem it solved
- **GPL v2 License** — The four freedoms of open source and how copyleft works
- **Ethics of Open Source** — Debate around monetization and community contribution
- **Linux Architecture** — Key directories, kernel modules, dependencies, and security
- **Linux vs Windows Server** — Comparison across cost, security, freedom, and usability

---

## Shell Scripts

| Script | Description |
|--------|-------------|
| Script 1 | Hello World — basic echo and script execution |
| Script 2 | Add Two Numbers — user input and arithmetic with `$(( ))` |
| Script 3 | Check if File Exists — `if-then-else` with `-f` flag |
| Script 4 | Print Current Date & Time — using `$date` |
| Script 5 | Interactive Menu — `while` loop + `case` statement with 5 system options |

---

## Key Takeaways

- Linux powers Android, cloud platforms (AWS/GCP/Azure), Docker, and more
- GPL v2's copyleft ensures all derivatives remain open source
- For servers and cloud infrastructure, Linux wins on cost, performance, and freedom
